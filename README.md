WordPress Vagrant Box
=====================
A [WordPress][WORDPRESS] instance provisioned via [Ansible][ANSIBLE] and [Vagrant][VAGRANT].

Usage
-----
From the command line run:

```sh
vagrant up
```

Then navigate to <http://localhost:8080> in your browser.

Shutting Down the Box
---------------------
From the command line run:

```sh
vagrant halt
```
